📊 SQL + Python Sales Analytics Pipeline

🚀 Project Overview

This project is an end-to-end data analytics pipeline that combines SQL
and Python to extract, analyze, and visualize business insights from
sales data.

🎯 Objectives

-   Build a structured data pipeline
-   Store and query data using SQL
-   Perform data cleaning and transformation
-   Generate visual insights
-   Apply a basic machine learning model

🧰 Tech Stack

-   Python
-   SQL (SQLite)
-   pandas
-   matplotlib
-   scikit-learn
-   Visual Studio Code

📂 Project Structure

sql-python-sales-analytics/ │ ├── data/ │ └── superstore.csv │ ├──
scripts/ │ ├── database.py │ ├── queries.py │ └── analysis.py │ ├──
outputs/ │ └── charts/ │ ├── requirements.txt └── README.md

⚙️ How to Run

1.  Install dependencies: python3 -m pip install -r requirements.txt

2.  Run pipeline: python3 scripts/database.py

3.  Run queries: python3 scripts/queries.py

4.  Run analysis: python3 scripts/analysis.py

📈 Features

-   Data pipeline
-   SQL analysis
-   Data visualization
-   Machine learning prediction
